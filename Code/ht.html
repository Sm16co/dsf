HTML Historik

HTML som är en förkortning för HyperText Markup Language och är en del av World Wide Web. HTML är ett språk som skapar möjlighet att beskriva ett dokuments struktur och innehåll. I det räknas rubriker, språk, styckesindelnings med mera (1).

Grundaren av HTML anses vara Tim Berners-Lee. Han var anställd på CERN och insåg snabbt att det var mycket svårt att dela information. Ofta behövdes det flera datorer för ett projekt då olika datorer lagrade olika information. Han kom på lösningen som blev programmet Enquire som kunde lagra, jämföra personalens projekt, samt kontrollera ifall vissa nyckelord fanns i dokumentet. Programmet blev bara en del av företaget. Det gav honom idén till att utveckla den första HTML. Han beskrev HTML i dokumentet med namnet ”HTML Tags” i slutet av år 1991 (1,3). Grunden till HTML kommer från ett tidigare språk nämligen SGML. SGML och HTML var inte helt likadana och inom vissa delar var de helt olika. HTML anses vara en förenkling av SGML (1-3). Första versionen av HTML som utvecklades av Tim B-L fick aldrig ett versions nummer men kallas ofta ”1.0” (2).

Det dröjde till början av 1993 innan HTML fick färg och bilder. Det var efter att webbläsaren Mosaic skapade nya element som allmänheten började använda det. Det var även under 1994 som Netscape presenterade sin webbläsare Navigator (NN) och den var väldigt bra för de som hade trögare uppkoppling. Netscape presenterade även ett par nya HTML-element och då blev väldigt populär hos kunderna. Microsoft svarade med att skapa webbläsaren Internet Explorer (MSIE)(A). För att säkra att webben inte skulle dela sig i många olika sidor införde man år 1995 HTML 2.0. Det dröjde två år innan HTML 3 kom ut. Det blev ett rent fiasko och väldigt få använde den (2). Därför kom utvecklings och standardiseringsgruppen W3C som grundades av Tim Berners-Lee efter att han lämnade CERN, att utveckla versionen 3.2 som använde alla element från NN och MSIE som standard (1,2,4).

1997 kom HTML 4 ut som var den första HTML versionen som uppmuntrade till att skilja mellan CSS och HTML. Innan den så skrev man allt i HTML och man använde olika tricks för att till exempel skapa utrymme mellan olika delar av sidan. Nu kunde man skilja design från själva koden och HTML 4 definierades i tre olika versioner för att underlätta användningen av CSS. Den första var strikt med att enbart använda HTML, den andra tillät användning av dem men nedvärderade CSS elementen medan den sista tillät användning av alla element. Kort efter publiceringen kom versionen 4.01 som åtgärdade innehållande bugg i versionen. Ur HTML föddes XML som var mycket hårdare vid läsning av koden. Ur dem båda så föddes XHTML och XHTML 1.0 nådde rekommendationsstatus redan år 2000. Senare påbörjade även utvecklingen av XHTML 2.0 men då man skrotade många gamla element lyckades man aldrig. Detta ledde till att Mozilla, Apple och Opera (2004) bildade WHATWG som skulle utveckla HTML 5 istället för XHTML 2.0. Tillskillnad från de föregående HTML versionerna så kunde allmänheten nu mejla in sina tips för dess utveckling. Efter att utvecklingen av HTML 5 börjat sätta fart anslöt sig även W3C till processen. 2014 publicerades HTML 5. 2016 kom versionen 5.1 ut och 2017 kom 5.2 ut (2).

Dagens version är 5.3 och den kom ut 2018. Det finns vissa skillnader mellan Html 1 och HTML 5. Man kan till exempel lagra data lokalt eller ladda upp video eller audio filer. I HTML 5 har även nya element kommit och HTML 5 kan tillskillnad från HTML även korrigera felen bättre. HTML 5 ger även stöd till CSS och JavaScript tillskillnad från HTML (5–7).

Ifall vi kollar på internet så påverkar den både miljön och samhället men även individen. Internet har påverkat individen så att kommunikationen blir bättre, vi kan även använda internet för att navigera eller roa oss. Men den kan även leda till depression och att olika terrororganisationer organiserar sig. Den påverkar samhället genom att man via internet kan öka handeln eller effektiviserar produktionen. Den kan även påverka samhället negativt då samhället kan utsättas för cyberattacker eller att vissa företag utsätts för piratkopiering. Miljön påverkas genom att det blir fler onlinemöten som sparar resor som skulle påverka miljön eller att medveten ökar då vi via internet kan läsa om miljöns påverkan. Men miljön påverkas även negativt när produktionen ökar och vi slänger saker (kasserar) men även då stora servrar behöver stora mängder energi för att vara så effektiva.
