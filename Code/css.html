CSS Historik

CSS eller Cascading Style Sheets, är ett språk som används för att forma eller strukturera olika texter skrivna i andra programmeringsspråk så som Html. Den används bland annat för färg, marginal, layout med mera (1).

CSS blev först utvecklad av Håkon Wium Lie som var en norsk webbutvecklare. Han kom på idén medan han jobbade tillsammans med bland annat Tim Berners-Lee (Html grundaren) (2). Han föreslog CSS 1994 (2) och det blev det vinnande förslaget som sedan även kom ut 1996 som en standard (3). CSS lanserades eftersom det var ganska svårt att genom ett och samma språk, alltså Html, kunna skriva både text, generera webbplatsen samt designa den (3).

Den första versionen av CSS kom ut 1996 i samarbete med W3C och även om rivalen Netscape Communication Corporation publicerade ett annat stilspråk så blev CSS vinnaren tack vare stöd från bland annat Bert Bos (Medgrundare i CSS) (1). De första webbläsarna som var kompatibel med CSS var Mac os och MSIE 5.0(3). CSS fortsatte sedan att utvecklas i en arbetsgrupp som styrdes av W3C för att skapa olika rekommendationer av CSS användande (1). CSS 2.0 blev en rekommendation för webbläsarna men snabbt insåg man att CSS krånglade och kort därefter släppte man CSS 2.1 för att korrigera det. CSS 3 blev den första som delades i olika moduler. Många moduler utvecklades och nu har vi ett tiotal av dem. Alla moduler är inte redo att användas fullständigt och är under ständig utveckling medan andra används för fullt. Olika moduler kan sedan ha olika versioner (3). Det dröjde till 2003 innan CSS blev det riktiga designspråket. Jeffrey Zeldmans skrev sin första bok om CSS år 2003 och det ledde till en större förståelse hos webbutvecklarna och kan räknas även som den revolutionerande boken inom CSS (3). Sista versionen som är ute är CSS 3. Även om vissa delar av CSS 4 (i en modul) lanseras är det inte versionen helt klart utan det finns små delar som är tillräckligt stabila för att vara med i version 3 (1,5).

Det är stor skillnad mellan CSS 1 och den nuvarande uppdateringen nämligen CSS 3. CSS 3 har tillskillnad från CSS 1 flera modifikationer men den största skillnaden från CSS 1 är att CSS 3 har olika moduler som gör att det är lättare att specialiseras på en viss del av CSS-en. Det gör även att varje modul är oberoende från en annan och att ingen webbläsare stödjer alla moduler då de är i ständig utveckling. CSS3 kan användas baklänges medan CSS inte kan använda CSS3 element. Även 3D element och färger är bättre i CSS3 än i CSS och är betydligt fler (3,6).

Fördelar med CSS är att man kan ändra hela stilen genom att bara ändra CSS och inte behöva ändra något annat språk. Du kan även till exempel göra så att alla h1 i texten blir i en viss färg eller storlek. Den största skillnaden är att webbsidan får olika utseende i olika webbläsare (7).

First Browser War som varade mellan 1995–2001 var ett webbläsarkrig mellan Microsoft (Mosaic) och Netscape. Det började med att Netscape lanserade sin webbläsare Netscape Navigator (NN) som var bättre för alla långsamma analoger. Netscape införde nya element i Html språket och då skapade ett problem för de som inte använde NN då de inte kunde komma in på vissa hemsidor. Netscape skapade även Javascript och lade till olika animeringar i Html som ledde till att den blev väldigt populär. Men när Netscape annonserade att de inte skulle behöva ett operativsystem så rasade Microsoft på grund av att deras operativsystem Windows inte mer skulle användas sattes alla resurser för att konkurrera ut NN. Man skapade då Internet explorer och det ledde till att personer som skapade en webbplats i NN hade svårt att anpassa den i Internet explorer och vice verse (3,8).
